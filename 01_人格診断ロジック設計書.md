## 統合モデル

共通コア（企業版はここまで）

Persona Score = タイプコンパス(30%) + BigFive(50%) + Enneagram(20%)

占い版（共通コア＋九星気学を追加した別計算）

## タイプコンパス変換（ユング理論ベースの4指標）

> ユーザー向け名称は「タイプコンパス」。裏側のロジックはユング心理学の4指標（いわゆるMBTI理論）そのものだが、公式MBTI設問は使わず独自設問で実装するため、この名称でブランディングする（詳細は[Human_Persona_Platform_企画設計書.md](Human_Persona_Platform_企画設計書.md)「1.5 差別化戦略」参照）。DBのテーブル名`mbti_results`・カラム名`ei/sn/tf/jp`は内部識別子としてそのまま使う。
>
> **実装済み**：[src/lib/type-compass.ts](src/lib/type-compass.ts) に独自設問24問（4軸×6問）と採点ロジックを実装済み。テストは[src/lib/type-compass.test.ts](src/lib/type-compass.test.ts)（`npm test`）。

  E/I    社交性   内省性

## 九星気学（占い版のみ）

### 本命星の算出

生年月日のみで一意に計算できる（追加の設問が不要）。

``` text
1. 立春（2/4）より前生まれは前年扱いにする
2. 西暦年を1桁になるまで各桁を足す（例：1990 → 1+9+9+0=19 → 1+9=10 → 1+0=1）
3. 11 - (2で出た数) = 本命星番号
4. 番号が10以上になった場合は9を引く
5. 番号 → 九星（1:一白水星 2:二黒土星 3:三碧木星 4:四緑木星 5:五黄土星 6:六白金星 7:七赤金星 8:八白土星 9:九紫火星）
```

月命星は本命星と生まれ月（節入り基準）から算出する。

> **既存資産の流用**：`bodyclystal/lib/kyusei.ts`（別クライアント向けに実装済み）に、本命星・月命星・日命星・傾斜宮まで含む検証済みロジックが既にある。本命星の算出式（年の桁和→`mod9(11 - s)`）はこの設計書と一致しており、Phase3実装時はゼロから作らずここを移植すればよい。日命星はユリウス通日ベースで実装・検証済み（1981/5/22=四緑木星で確認）。HPPでは本命星＋五行のみ使う設計だが、将来的に月命星・傾斜宮まで拡張する場合も流用できる。

### 五行対応

  九星                     五行
  ------------------------ ------
  一白水星                 水
  二黒土星・五黄土星・八白土星   土
  三碧木星・四緑木星       木
  六白金星・七赤金星       金
  九紫火星                 火

## 相性ロジック

共通コア：相性 = 価値観40% + 性格30% + 動機30%

占い版：上記に加えて五行相性を±10点の補正として加える

``` text
相生（高め合う）：木→火→土→金→水→木
相剋（打ち消し合う）：木⇔土、土⇔水、水⇔火、火⇔金、金⇔木
```

-   相生の関係：+10点
-   相剋の関係：-10点
-   それ以外（同じ五行 or 無関係）：±0点

80点以上：非常に良い 60点以上：良い 40点以上：注意
